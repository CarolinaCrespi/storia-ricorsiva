# C'era una volta un re…

Un'esperienza narrativa interattiva ispirata alla filastrocca ricorsiva italiana.

La storia comincia. La storia non finisce mai.

## Di cosa si tratta

*C'era una volta un re, seduto sul sofà, che disse alla sua serva: "Raccontami una storia". E la storia cominciò così…*

Questa filastrocca è un esempio antico di **ricorsione**: una struttura che si richiama da sola all'infinito, come una funzione senza caso base. Secoli prima dei computer, i bambini conoscevano già i loop infiniti.

Il progetto la trasforma in un'esperienza visiva e sonora: un libro animato che si apre, si sfoglia, e poi — quando sembra finita — ricomincia dall'interno, ancora e ancora, sprofondando in se stesso.

## Funzionalità

- Libro animato con pagine che si girano in 3D
- Illustrazioni SVG originali per ogni scena
- Musica procedurale generata via Web Audio API (scala lidia, arpa + flauto)
- Atmosfera con candele, particelle e luce ambientale
- Ricorsione infinita: ogni storia contiene la storia successiva
- Zero dipendenze esterne — un solo file HTML

## Stack

HTML · CSS · JavaScript · Web Audio API · SVG

## Ispirazione

La filastrocca esiste in varianti simili in molte culture e lingue.
La struttura ricorsiva è la stessa ovunque: un re, una serva, un libro, e una storia che non ha mai intenzione di finire.
